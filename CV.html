Optimizing Spectral Signature in Code Backdoor Detection
Quoc Hung Le, Thanh Le-Cong, Bach Le, Bowen Xu
TL;DR: In this paper, we revisit the applicability of Spectral Signature-based defenses in the context of backdoor attacks on code models. We explored the impact of different settings of the Spectral Signature. We discovered a new proxy metric that can more accurately estimate the actual performance of Spectral Signature without model retraining after the defense.
